# BPTtoECGRecon
This project includs several datasets for the paper "Deep learning-driven contactless ECG in MRI via beat pilot tone for motion-resolved image reconstruction and heart rate monitoring.

Paired BPT and ground truth signal dataset for model trainning can be downloaded via following Link:

https://drive.google.com/drive/folders/1KO18NHPWtzWqRqwxEKRbBwwWV2Szzpz9?usp=sharing

In the folder, BH stands for 'Breath Hold' and FB stands for 'Free Breathing'.
